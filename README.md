# Baahumanyu One Gram Jewellery Website

A professional, modern e-commerce website for Baahumanyu One Gram Jewellery built with HTML, CSS, and JavaScript.

## 📋 Features

✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
✅ **Product Catalog** - Browse jewelry by category (Necklaces, Earrings, Bangles, Vaddanam, Rings, Pendants)
✅ **Shopping Cart** - Add/remove items, view cart summary with tax calculation
✅ **Product Details** - Modal view for detailed product information
✅ **Contact Section** - Easy contact form and business information
✅ **WhatsApp Integration** - Direct checkout via WhatsApp
✅ **Local Storage** - Cart persists even after page refresh
✅ **Modern UI** - Gold & Brown color scheme reflecting jewelry elegance

## 🎨 Design Features

- **Color Scheme**: Gold (#D4AF37) + Brown (#8B6F47) - Premium jewelry aesthetic
- **Typography**: Clean, modern fonts for readability
- **Images**: High-quality product images with smooth hover effects
- **Navigation**: Sticky navigation bar with mobile hamburger menu
- **Sections**: Hero, About, Categories, Products, Cart, Contact, Footer
